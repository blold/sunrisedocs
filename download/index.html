如何下载及更新

这里讲解如何下载及更新cms系统。

下载地址

建议使用git clone 命令来下载，也可以直接去以下链接直接下载

https://github.com/blold/sunriseCMS

git clone git@github.com:blold/sunriseCMS.git

更新cms

会在未来不定时更新各个组件， 若是使用git clone 来下载的，可以使用git pull 来在未来直接进行合并更新。

项目根目录，命令行

git pull
